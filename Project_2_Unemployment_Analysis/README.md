# Unemployment Analysis with Python

## Objective
To analyze unemployment rate data and observe trends over time, especially during the COVID-19 period.

## Steps Performed
1. Loaded dataset using pandas
2. Cleaned column names
3. Removed missing values
4. Converted Date column to datetime
5. Generated summary statistics
6. Visualized unemployment trend over time
7. Analyzed COVID impact
8. Compared average unemployment across regions

## Key Insight
A sharp increase in unemployment rate was observed during the COVID-19 period.
