# Iris Flower Classification

## Objective
To build a machine learning model that classifies iris flowers into three species based on their measurements.

## Dataset
Used the built-in Iris dataset from Scikit-learn.

## Algorithm Used
Logistic Regression

## Steps Performed
1. Loaded dataset
2. Split data into training and testing sets
3. Trained the model
4. Evaluated accuracy

## Result
The model achieved 100% accuracy on the test dataset.
